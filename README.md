# Grocery Store Web App (Django)

A web application built with Django that allows users to browse and purchase food items from local vendors.

## Features
- User authentication (login and signup)
- Product listing and categorization
- Shopping cart system
- Order management

## Tech Stack
- Python (Django)
- SQLite
- HTML/CSS

## Project Purpose
This project was built to explore how local food vendors can digitize their sales through an online platform.

## Installation

1. Clone the repository:
   git clone https://github.com/barnabz/lapulla_store.git

2. Navigate into the project:
   cd lapulla_store

3. Install dependencies:
   pip install -r requirements.txt

4. Run the server:
   python manage.py runserver
